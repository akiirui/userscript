# 直播彈幕控制

自動關閉直播彈幕。

### 支持的直播平臺

- 鬥魚直播 `www.douyu.com`
- 虎牙直播 `www.huya.com`
- YY 直播 `www.yy.com`

#### 臨時損壞的平臺

- 嗶哩嗶哩直播 `live.bilibili.com`

  > 當前「**彈幕控制按鈕**」只在鼠標懸停在播放器上時才創建，且在移出時被刪除。
