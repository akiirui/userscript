# Wikipedia Redirector

Automatically redirect Wikipedia language variants

Redirect `/wiki/` and these language variants:

- `zh-HK`
- `zh-MO`
- `zh-TW`
- `zh-SG`

To `zh-CN`.

Compatible with mobile devices.
